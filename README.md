# HR Analytics Dashboard (Tableau)

## Project Overview
This project analyzes human resources (HR) data using Tableau to understand workforce distribution, employee performance trends, and key organizational metrics.

The dashboard is designed to provide a clear and interactive view of HR performance indicators that can support strategic workforce planning and management decisions.

---

## Why This Project?
I created this project to practice building business-oriented dashboards that focus on clarity, usability, and actionable insights.

The goal was to design a dashboard that HR teams and management can use to quickly understand workforce metrics and trends.

---

## Business Objectives
- Analyze employee distribution across departments
- Monitor performance metrics and workforce trends
- Identify attrition or retention patterns
- Evaluate workforce demographics
- Support data-driven HR decision-making

---

## Tools & Technologies
- **Tableau**
  - Interactive dashboards
  - Filters and drill-down analysis
  - KPI visualizations
- **Excel / CSV** (data source)

---

## Dataset Information
- **Source:** Company-like HR dataset
- **File:** `HR Dashboard.twbx`
- **Data Includes:**
  - Employee ID
  - Department
  - Job Role
  - Salary or Compensation
  - Performance Rating
  - Experience / Tenure
  - Attrition Status
  - Location
